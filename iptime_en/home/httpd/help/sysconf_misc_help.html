• 공유기 이름
    • 공유기 이름을 설정하면, IE등의 웹브라우저 상단의 타이틀 바에 이름이 표시됩니다.
    • 공유기 이름은 E-mail 보고서의 제목에 표시됩니다.
  • 자동 설정 저장
    • 자동 설정 저장 기능을 '실행'시에는 모든 설정이 자동으로 저장됩니다.
    • 자동 설정 저장 기능을 '중단'시에는 설정 시에는 저장이 되지 않으며, 우측 상단의 [설정 저장] 버튼을 누를 경우 모든 설정이 저장됩니다.

  • 설정화면 자동연결
    • 인터넷 끊김시 공유기 설정화면으로 자동 연결되는 기능입니다.

  • 로그인 페이지 설정
    • 설정 화면 접속시 로그인 페이지를 보여줄지를 설정합니다.

  • 관리도구 접속방법
    • 로그인 페이지에서 관리도구를 클릭하였을때 접속 방법을 설정합니다.

  • UPNP 설정
    • UPNP 기능을 동작시킬 수 있습니다.
    • UPnP(Universal Plug and Play)는 특히, 가정 내에서 PC와 인터넷공유기를 피어-투-피어 방식의 네트워크로 연결하기 위해 보편적으로 사용되는 아키텍처입니다.
    • UPNP 기능을 사용하시면 MSN 메신저의 모든 기능을 공유기환경에서 사용하실 수 있습니다.
    • [UPNP 포트포워딩 리스트] 버튼을 누르면 , UPNP프로토콜에 의하며 설정된 포트포워딩 리스트를 볼 수 있습니다.

  • 공유기 재시작
    • 공유기가 재시작 됩니다.
    • 저장하지 않은 설정값은 지워질 수 있습니다.
