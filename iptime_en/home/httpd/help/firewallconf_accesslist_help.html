• 외부 접속 보안
    • 허가되지 외부 단말이 공유기에 접속하는 것을 차단하는 기능입니다.
    • 공유기의 접속은 웹 브라우저에 주소란에 http://xxx.xxx.xxx.xxx:pppp와 같이 URL을 입력하면 공유기 설정 화면에 접속이 가능합니다.
  • 원격 관리 포트 사용
    • 외부 접속을 위해서는 반드시 원격 관리 포트 번호를 지정하셔야 합니다.
  • 외부 접속 보안 사용
    • 외부 접속 보안 사용을 체크하면 등록된 IP주소에서만 원격 관리 포트 번호를 통해서 공유기 설정 웹 서버로 접속 할 수 있습니다.
  • IP주소 및 설명 추가
    • 외부 접속 보안 동작시 추가한 외부 IP 주소만 공유기 접속을 허용합니다.


  • 내부 접속 보안
    • 내부에서의 공유기 설정 웹 서버 접속은 기본적으로 공유기와 연결된 모든 PC에서 접속이 가능합니다.
    • 내부 접속 보안을 사용을 체크하면 등록된 IP 주소에만 공유기 설정 웹 서버로 접속 할 수 있습니다.
