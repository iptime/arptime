• 연결 상태
    • 인터넷 연결 상태를 보여 줍니다.
    • 인터넷 연결이 완료되면 '외부 IP 주소', '서브넷마스크', '기본 게이트웨이', '기본 DNS 서버', '보조 DNS 서버' 항목이 설정됩니다.
  • 연결 방식
    • 인터넷 연결 방식을 보여 줍니다.
  • 외부 IP 주소
    • 인터넷 서비스회사로 부터 부여받은 공인 IP 주소 입니다.
    • 내부에서 각종 서버를 운용할 경우에는 해당 IP 주소를 통해 접속해야 합니다.
    • 인터넷 사용을 위해서는 반드시 설정이 되어 있어야 하는 항목입니다.
  • 서브넷마스크
    • 인터넷 서비스회사에서 설정한 논리적인 네트워크 그룹을 나타내는 항목입니다.
    • 인터넷 사용을 위해서는 반드시 설정이 되어 있어야 하는 항목입니다.
  • 기본 게이트웨이
    • 인터네 사용시 반드시 경유하게 되는 인터넷 서비스회사의 라우터 IP 주소를 나타내는 항목입니다.
    • 인터넷 사용을 위해서는 반드시 설정이 되어 있어야 하는 항목입니다.
  • 기본 DNS 서버
    • 인터넷에서 도메인이름을 확인하기 위한 데이타베이스 부분에 대한 정보를 갖고 있는 서버입니다.
    • 인터넷 사용을 위해서는 반드시 설정이 되어 있어야 하는 항목입니다.
  • 보조 DNS 서버
    • '기본 DNS 서버'에 문제가 발생할 경우 필요한 백업용 서버 입니다.
    • 통신 회사에 따라 설정되지 않을 수도 있습니다.
  • MAC 주소
    • 공유기의 MAC주소를 보여 줍니다.
